Vibe 参考图隐私处理

选择 NovelAI Vibe 文件后，页面会在本地替换预览图字段，生成不暴露原参考图的副本。

拖入 .naiv4vibe / JSON 文件，或点击选择只替换 image、thumbnail 等预览字段；不会上传文件，也不会改动 Vibe encoding。
等待选择文件。

处理范围

  • 支持顶层或嵌套的 image、thumbnail、preview、previewImage。
  • image 会输出裸 Base64，thumbnail 会输出 Data URL，兼容常见 NovelAI Vibe 结构。
  • 保留原文件其它字段，包括 encodings、importInfo、name 与 id。
  • 生成文件名会自动追加 _private_preview。
